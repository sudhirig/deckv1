# Design System - Standardized Values

## Spacing Standard
- **Padding Small**: `p-2` (8px) - for small cards, inner elements
- **Padding Medium**: `p-3` (12px) - for standard cards, content boxes
- **Padding Large**: `p-4` (16px) - for major containers (rare)

## Margin Standard
- **Margin Small**: `mb-1` (4px) - for tight spacing
- **Margin Standard**: `mb-2` (8px) - for most elements
- **Margin Section**: `mb-3` (12px) - for section separators
- **Margin Large**: `mb-4` (16px) - for major sections (rare)

## Spacing Standard
- **Space-Y Small**: `space-y-1` (4px) - for tight lists
- **Space-Y Standard**: `space-y-2` (8px) - for most lists/grids
- **Space-Y Section**: `space-y-3` (12px) - for section spacing

## Gap Standard
- **Gap Small**: `gap-2` (8px) - for tight grids
- **Gap Standard**: `gap-3` (12px) - for most grids
- **Gap Large**: `gap-4` (16px) - for wide spacing (rare)

## Typography Standard
- **Body Text**: `text-xs` (12px) - for most content
- **Body Large**: `text-sm` (14px) - for emphasized content
- **Subtitle**: `text-base` (16px) - for subtitles
- **Title Small**: `text-lg` (18px) - for small titles
- **Title**: `text-xl` (20px) - for standard titles
- **Title Large**: `text-2xl` (24px) - for major titles
- **Title XL**: `text-3xl` (30px) - for hero titles (rare)

## Max Height Standard
- **Small Component**: `max-h-[40vh]` - for small charts
- **Standard Component**: `max-h-[50vh]` - for standard components
- **Large Component**: `max-h-[60vh]` - for large components
- **Full Component**: `max-h-[65vh]` - for full-screen components (rare)
